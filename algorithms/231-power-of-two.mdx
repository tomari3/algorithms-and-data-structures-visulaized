<h2>Introduction</h2>

Given an integer <code>n</code>, return <code>true</code> if it is a power of two. Otherwise, return <code>false</code>.

An integer <code>n</code> is a power of two, if there exists an integer <code>x</code> such that <code>n == 2<sup>x</sup></code>.

<h3>Examples</h3>

Example 1:

```
Input: n = 1
Output: true
Explanation: 20 = 1
```

Example 2:

```
Input: n = 16
Output: true
Explanation: 24 = 16
```

Example 3:

```
Input: n = 3
Output: false
```

<h3>Test Cases</h3>

```
1
16
3
```

<h3>Recursive Solution</h3>

```
/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfTwo = function (n) {
  if (n - 1 < 0) return false;
  return n === 0 || n === 1 ? true : isPowerOfTwo(n / 2);
};
```
